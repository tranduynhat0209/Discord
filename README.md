# Discord

## Backend

### Installation & Environment Setup
```
cd backend

npm i

cp .env.example .env

// you might see file .env generated, fill the required information about your database and the website in each field.
```

### Set up JWT keys

Run
```
./keygen.sh 
```

### Start the server
```
npm start
```
